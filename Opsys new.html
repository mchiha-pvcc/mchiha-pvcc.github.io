Operating System Terminology

Terminology Definition
Operating System Is the software that controls the overall operation of a computer. It provides the means by which a user can store and retrieve files.
Job queue A storage organization in which tasks are ordered.
FIFO abbreviation for first in first out. the objects are removed from the queue in the order in which the arrived.
Interactive processing A terminal consisted of little more than electronic typewriter where the user can type input and read the computer's response that was printed on paper.
Time-sharing An operating system that was design to provide service to multiple users at the same time.
Multiprogramming When the time is devided into intervals and then the execution of each job is restricted to only one interval at a time.
Application software is consisted of the programs for performing tasks particular to the machine's utilization.
System software provides the infrastructure that the application software requires, in much the same manner as a nation's infrastructure provides the foundation on which its citizens rely for their individual lifestyle.
Utility Software Consists of programs for performing activities that are fundamental to computer instalations but not included in the operating system.
Graphical user interface (GUI) where objects to be manupilated such as files and programs are presented pictorially on the display as icons.
Shell is an older user interface that communicates with users through textual messages using a keyboard and a monitor screen.
Kernel is the enternal part of an operating system.
Device Drivers Are the software units that communicates with the controllers.
File manager one of the kernal's units. it coordinates the use of the machine's mass storage facilities.
Memory manager another unit of the kernal's. it coordinates the machine's use of main memory.
Virtual memory is a large fictional memory space created by paging.
boot strapping is a procedure that provides the software infrastructure required by other software units.
Rom a small portion of a computer's main memory where the CPU expects to find its intial program is constructed from special nonvolatile memory cell.
Deadlock any situation in which no member of some group of entities can proceed because each waits for another member, including itself, to take action.
Spooling is a technique of holding data for output at a later but more convenient time.
Super user (or administrator) is the person who gains highly privileged access to the operating system.
Multi-factor authentication An authentication system that requires more than one distinct authentication factor for successful authentication.
